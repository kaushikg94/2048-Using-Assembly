2048 Using Assembly

Gokul Kaushik    Henry Chen

An implementation of the popular game 2048 using Assembly code on the Nios-II processor.

Background Info

  • Original web game created by Gabriele Cirulli is playable here.
  • Submission for the project component of ECE243: Computer Organization course.
  • Made using only Assembler code and C.

Gameplay Tutorial

Try playing the original game first.

  • There is a 4-by-4 (16 block) grid space and you start with two blocks of 2.
  • The possible keyboard moves are up, down, left and right.
  • Blocks will either shift or merge with an adjacent block of equal value.
  • Every time a move is made, a random value block will be added into an available empty slot.
  • The goal is to repeatedly add blocks of equal value till you reach the 2048 block.
  • The game is lost when there are no possible merges, no available slots, and no 2048 block.
