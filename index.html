Gameplay Tutorial

To have a better understanding of the game, you can play Gabriel Cirulli's version here.

  • The game consists of a 4-by-4 grid (16 tiles in total) of blocks.
  • Tiles are empty initially and the game starts with a tile with the number 2.
  • You can use the arrow keys: up, down, left and right to move the tile in the respective direction accross the grid.
  • For every move made, the game introduces a new number tile into a random grid.
  • Usually the starting point is on the lowst empty row and the number is 2. However, as the game progresses, tiles of increasing number are introduced.
  • When a key is pressed in any directions, tiles of equal value are merged to form their sum. The basic tile is 2 and all tiles merged form a value that is a power of 2.
  • The victory condition is to keep merging tiles of equal value until you reach 2048 (hence the name).
  • The game is lost when the 16 block grid is full and there can be no possible merges (resulting in a stagnant grid).
